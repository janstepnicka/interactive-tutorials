코드 내성은 클래스, 함수 및 키워드를 검사하여 그것들이 무엇인지, 무엇을 하는지, 무엇을 알고 있는지 파악할 수 있는 능력입니다.

Python은 코드 내성을 위한 여러 함수와 유틸리티를 제공합니다.

    help()
    dir() 
    hasattr() 
    id() 
    type() 
    repr() 
    callable() 
    issubclass() 
    isinstance() 
    __doc__ 
    __name__ 
    

가장 중요한 것은 종종 help 함수인데, 다른 함수가 무엇을 하는지 찾는 데 사용할 수 있기 때문입니다.
