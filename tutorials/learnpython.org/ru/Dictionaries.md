A dictionary — это тип данных, похожий на массивы, но работающий с ключами и значениями вместо индексов. Каждое значение, хранящееся в словаре, может быть доступно с помощью ключа, который может быть любым типом объекта (строка, число, список и т.д.) вместо использования его индекса для обращения.

Например, база данных номеров телефонов может храниться с использованием словаря следующим образом:

    phonebook = {}
    phonebook["John"] = 938477566
    phonebook["Jack"] = 938377264
    phonebook["Jill"] = 947662781
    print(phonebook)

Кроме того, словарь может быть инициализирован с такими же значениями в следующей нотации:

    phonebook = {
        "John" : 938477566,
        "Jack" : 938377264,
        "Jill" : 947662781
    }
    print(phonebook)

### Итерация по словарям

Словари могут быть перебраны, как и список. Однако словарь, в отличие от списка, не сохраняет порядок хранимых в нем значений. Чтобы итерировать по парам ключ-значение, используйте следующий синтаксис:
    
    phonebook = {"John" : 938477566,"Jack" : 938377264,"Jill" : 947662781}
    for name, number in phonebook.items():
        print("Phone number of %s is %d" % (name, number))

### Удаление значения

Чтобы удалить указанный индекс, используйте одну из следующих нотаций:
    
    phonebook = {
       "John" : 938477566,
       "Jack" : 938377264,
       "Jill" : 947662781
    }
    del phonebook["John"]
    print(phonebook)

или:
    
    phonebook = {
       "John" : 938477566,
       "Jack" : 938377264,
       "Jill" : 947662781
    }
    phonebook.pop("John")
    print(phonebook)
